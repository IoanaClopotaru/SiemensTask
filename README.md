# Siemens Technical Assessment

This repository contains my solutions for the Siemens Technical Assignment.  
The project is divided into two main problems, covering system design and software implementation.

---

## Requirements

Ensure the following tools are installed before running the project:

- .NET 8 SDK (or compatible version)
- Mermaid viewer (for Problem 1 diagrams)

---

## Check Installed .NET Version

```bash
dotnet --version
```

---

## Viewing Diagrams (Problem 1)

You can open Mermaid diagrams using:

- Visual Studio Code with a Mermaid extension
- GitHub built-in Markdown preview
- Mermaid Live Editor: https://mermaid.live
